Chromium: CVE-2023-4863 Heap buffer overflow in WebP

Revision: 1.0 (NEW)
Published: 12-09-2023
Official Fix:
Yes
Product family info: microsoft edge (chromium-based)
Show More
Build numbers:
116.0.1938.81
KB Articles:
  • Release Note (12-09-2023)
Update Packages:
  • Edge Update
Summary: Microsoft Edge (Chromium-based) is affected by a heap buffer overflow vulnerability in WebP, identified as CVE-2023-4863. This vulnerability was assigned by Chrome and is ingested by Microsoft Edge through Chromium, which addresses the issue. An exploit for CVE-2023-4863 exists in the wild, making it crucial for system administrators to take immediate action. The vulnerability is caused by a flaw in the Chromium Open Source Software (OSS) consumed by Microsoft Edge. An attacker could exploit this vulnerability to execute arbitrary code, posing a severe risk to affected systems. Microsoft has provided a security update for Microsoft Edge (Chromium-based) with the build number 116.0.1938.81, which system administrators are strongly advised to apply to mitigate the risk of exploitation. For more information, refer to the Google Chrome Releases page. Additionally, affected customers can check if the update is installed using PowerShell command: Get-AppxPackage -Name Microsoft.WebpImageExtension. It's important to note that Microsoft Store will automatically update affected customers, and those in disconnected environments can obtain the update through their organizations using the Microsoft Store for Business and Microsoft Store for Education.
