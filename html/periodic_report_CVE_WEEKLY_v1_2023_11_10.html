Summary: Windows Cryptographic Services is affected by an information disclosure vulnerability (CVE-2023-36907) with a base score of 5.5 and a temporal score of 4.8. The vulnerability allows a locally authenticated attacker to read small portions of heap memory by sending a specially crafted request to the cryptography provider's vulnerable function. The attack vector is local, with low complexity and privileges required, and no user interaction. The impact is high on confidentiality, but there is no impact on integrity and availability. Microsoft has provided a complete vendor solution, and security updates are available for various Windows Server and Windows client versions. System administrators are advised to apply the relevant security updates to mitigate the risk of exploitation. For more information, refer to the Microsoft Support Lifecycle page.

Summary: Microsoft Edge (Chromium-based) is affected by a use after free vulnerability in WebAudio, identified as CVE-2023-5996. This vulnerability was assigned by Chrome and is addressed by ingesting Chromium. The latest version of Microsoft Edge (Chromium-based) with build numbers 119.0.2151.58 and 118.0.2088.102 is no longer vulnerable.
