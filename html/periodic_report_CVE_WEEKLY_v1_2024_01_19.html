CVE-2024-21337 Microsoft Edge (Chromium-based) Elevation of Privilege Vulnerability

Official Fix:
Information update only
Product family info: microsoft edge (chromium-based)
Show More
Build numbers:
120.0.2210.133
KB Articles:
  • Release Note (11-01-2024)
Update Packages:
  • Edge Update
Summary: Microsoft Edge (Chromium-based) is affected by an elevation of privilege vulnerability identified as CVE-2024-21337. This vulnerability has a base score of 5.2/10 and a temporal score of 4.5/10 according to the CVSS 3.1 metrics. The attack vector is local, requiring user interaction, and the attack complexity is high. Successful exploitation could lead to a browser sandbox escape, with potential minor impact on confidentiality, integrity, and availability. Microsoft has released security updates for Microsoft Edge (Chromium-based) versions 120.0.2210.133, 120.0.2210.160, and 120.0.6099.216/217 to address this vulnerability. System administrators are advised to apply these updates to mitigate the risk of exploitation. The vulnerability is not directly exploitable and would require an attacker to combine it with other vulnerabilities to perform an attack. For more information, refer to the Microsoft Security Response Center page.

KB Articles:

Update Packages:
  • Security Update (KB5032192)
  • Security Update (KB5032190)
  • Security Update (KB5032189)
Summary: Microsoft has addressed the CVE-2023-24023 Bluetooth vulnerability, impacting Windows 10, 11, and Server 2019/2022. The vulnerability allows spoofing attacks and is related to the use of BR/EDR Secure Connections. Microsoft released a software update enforcing the use of specific encryption and authentication algorithms for Bluetooth pairings that have used BR/EDR Secure Connections. This update aims to mitigate the vulnerability by ensuring the use of secure connections for subsequent Bluetooth pairings. It is crucial to note that the new BR/EDR Secure Connections algorithms will only be utilized if both the local system and the remote paired device support BR/EDR Secure Connections. Additionally, increasing the minimum encryption key size, as per Windows guidance, is advised to enhance security. The exploitability assessment indicates that the vulnerability is less likely to be exploited. System administrators are urged to apply the security updates for the affected Windows versions to safeguard against potential spoofing attacks. For more information, refer to the Microsoft Security Updates page.
