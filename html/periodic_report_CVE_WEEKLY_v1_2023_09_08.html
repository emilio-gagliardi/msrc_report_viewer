Summary: Microsoft has updated the links to the Windows Update Catalog for the CVE-2023-24872 Microsoft PostScript and PCL6 Class Printer Driver Remote Code Execution Vulnerability. This is an informational change only. The vulnerability allows remote code execution and has a CVSS score of 8.8. An authenticated attacker with normal privileges could send a modified XPS file to a shared printer, resulting in remote code execution. The attack vector is network-based, with low attack complexity and no user interaction required. The impact includes high confidentiality, integrity, and availability. Microsoft has provided security updates for various Windows versions, including Windows 10, Windows 11, and Windows Server, addressing the vulnerability with build numbers 10.0.22621.1413, 10.0.19044.2728, 10.0.20348.1607, and others. System administrators are advised to apply the relevant security updates to mitigate the risk of exploitation. For more information, refer to the Microsoft Support Lifecycle and the Windows Update Catalog.

Chromium: CVE-2023-4764 Incorrect security UI in BFCache

Revision: 1.0 (NEW)
Published: 07-09-2023
Official Fix:
Yes
Product family info: microsoft edge (chromium-based)
Show More
Build numbers:
116.0.1938.76
KB Articles:
  • Release Note (07-09-2023)
Update Packages:
  • Edge Update
Summary: Microsoft Edge (Chromium-based) is affected by CVE-2023-4764, an incorrect security UI vulnerability in BFCache. This vulnerability was assigned by Chrome and is addressed by ingesting Chromium. The vulnerability is caused by a flaw in the Chromium Open Source Software (OSS) consumed by Microsoft Edge. An attacker could exploit this vulnerability to execute arbitrary code in the context of the current user. Microsoft has provided a solution by releasing a security update for Microsoft Edge (Chromium-based) with the build number 116.0.1938.76. System administrators are advised to update their Microsoft Edge installations to the latest version to mitigate the risk of exploitation. For more information, refer to the Google Chrome Releases page.
