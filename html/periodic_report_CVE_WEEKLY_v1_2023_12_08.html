Chromium: CVE-2023-6512 Inappropriate implementation in Web Browser UI

Revision: 1.0 (NEW)
Published: 07-12-2023
Official Fix:
Yes
Product family info: microsoft edge (chromium-based)
Show More
Build numbers:
120.0.2210.61
KB Articles:
  • Release Note (07-12-2023)
Update Packages:
  • Edge Update
Summary: Microsoft Edge (Chromium-based) is affected by CVE-2023-6512, an inappropriate implementation in the Web Browser UI, assigned by Chrome. This vulnerability is in the Chromium Open Source Software (OSS) consumed by Microsoft Edge. An attacker could exploit this vulnerability to execute arbitrary code in the context of the current user. Microsoft has addressed this vulnerability by ingesting Chromium, and the latest version of Microsoft Edge (Chromium-based) with build number 120.0.2210.61 is no longer vulnerable. System administrators are advised to update their Microsoft Edge installations to the latest version to mitigate the risk of exploitation. For more information, refer to the [Google Chrome Releases](https://chromereleases.googleblog.com/2023) page.

Summary: Microsoft Edge (Chromium-based) is affected by an Information Disclosure Vulnerability identified as CVE-2023-38174. The vulnerability has a low severity with a CVSS score of 4.3. It requires user interaction, where the user needs to click on a specially crafted URL to be compromised by the attacker. CVE-2023-35618 Microsoft Edge (Chromium-based) Elevation of Privilege Vulnerability

Revision: 1.0 (NEW)
Published: 07-12-2023
Official Fix:
Yes
Product family info: microsoft edge (chromium-based)
Show More
Build numbers:
120.0.2210.61
KB Articles:
  • Release Note (07-12-2023)
Update Packages:
  • Edge Update
Summary: Microsoft Edge (Chromium-based) is affected by an Elevation of Privilege Vulnerability, identified as CVE-2023-35618, with a CVSS score of 9.6/8.3. The vulnerability allows an attacker to gain the privileges needed to perform code execution, potentially leading to a browser sandbox escape. The attack vector is through a web-based scenario, where an attacker hosts a website containing a specially crafted file designed to exploit the vulnerability, requiring user interaction to click a link and open the file. The severity is rated as Moderate due to the amount of user interaction required, which downgrades the severity according to Microsoft's guidelines. Microsoft has provided an official fix by releasing a security update for Microsoft Edge (Chromium-based) with the build number 120.0.2210.61. System administrators are advised to update their Microsoft Edge installations to this version to mitigate the risk of exploitation. For further details, refer to the Microsoft Security Response Center page for CVE-2023-35618.
