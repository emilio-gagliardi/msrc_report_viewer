Chromium: CVE-2023-4368 Insufficient policy enforcement in Extensions API

Revision: 1.0 (NEW)
Published: 21-08-2023
Official Fix:
Yes
Product family info: microsoft edge (chromium-based)
Show More
Build numbers:
116.0.1938.54
KB Articles:
  • Release Note (21-08-2023)
Update Packages:
  • Edge Update
Summary: Microsoft Edge (Chromium-based) is affected by CVE-2023-4368, an insufficient policy enforcement vulnerability in the Extensions API. This vulnerability, assigned by Chrome, is addressed by ingesting Chromium, and the latest version of Microsoft Edge (Chromium-based) is no longer vulnerable. The underlying cause is a flaw in the Chromium Open Source Software (OSS) consumed by Microsoft Edge. An attacker could exploit this vulnerability to bypass extension policy enforcement and potentially execute arbitrary code. Microsoft has provided a solution by releasing a security update for Microsoft Edge (Chromium-based) with the build number 116.0.1938.54. System administrators are advised to update their Microsoft Edge installations to the latest version to mitigate the risk of exploitation. For more information, refer to the Google Chrome Releases page.
