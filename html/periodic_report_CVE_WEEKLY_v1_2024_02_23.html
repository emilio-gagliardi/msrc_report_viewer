CVE-2023-36019 Microsoft Power Platform Connector Spoofing Vulnerability

Revision: 1.2
Published: 16-02-2024
Official Fix:
Information update only
Product family info:
Show More
Build numbers:
Update Packages:
Summary: Microsoft Power Platform Connector is impacted by a critical spoofing vulnerability, CVE-2023-36019, with a CVSS score of 9.6. The vulnerability allows an attacker to manipulate a malicious link, application, or file to disguise it as legitimate, tricking the victim. It requires user interaction, where the user must click on a specially crafted URL to be compromised. Successful exploitation could lead to a scope change, with malicious scripts executing in the victim’s browser. Microsoft has provided a mitigation by requiring custom connectors using OAuth 2.0 to use a per-connector redirect URI, with a deadline of March 29, 2024. After this date, users will no longer be able to create connections to or use existing OAuth 2.0 custom connectors that have not been updated. Affected customers were notified via the Microsoft 365 Admin Center and Service Health in the Azure Portal. System administrators are advised to validate and update their custom connectors to mitigate the risk of exploitation.

CVE-2024-21315 Microsoft Defender for Endpoint Protection Elevation of Privilege Vulnerability

Revision: 1.0 (NEW)
Published: 20-02-2024
Official Fix:
Information update only
Product family info:
Show More
Build numbers:
Update Packages:
Summary: Microsoft Defender for Endpoint Protection is affected by an elevation of privilege vulnerability identified as CVE-2024-21315, with a CVSS score of 7.8. The vulnerability allows an attacker to gain SYSTEM privileges. It was addressed by updates released in November 2023, but inadvertently omitted from the November 2023 Security Updates. Microsoft strongly recommends installing the November 2023 updates to mitigate this vulnerability. Various Windows versions and editions, including Windows Server and Windows 10, are impacted. Security updates with build numbers 5032202, 5032189, 5032190, 5034819, 5032197, 5034830, and others are available to address this issue. The impact of exploitation includes high confidentiality, integrity, and availability risks. The attack vector is local, with low attack complexity and privileges required. No user interaction is needed for exploitation. A complete vendor solution is available, and no exploit code is currently available, making exploitation less likely. System administrators are strongly advised to install the recommended updates to protect affected systems from this elevation of privilege vulnerability.

CVE-2024-20677 Microsoft Office Remote Code Execution Vulnerability

Revision: 2.1
Published: 23-02-2024
Official Fix:
Information update only
Product family info:
Show More
Build numbers:
Update Packages:
Summary: A remote code execution vulnerability, CVE-2024-20677, affects Microsoft Office, including Office 2019, Office 2021, Office LTSC for Mac 2021, and Microsoft 365. The vulnerability exists in FBX, leading to remote code execution, and has a CVSS base score of 7.8. The attack vector is local, requiring user interaction, and successful exploitation could result in total loss of confidentiality, integrity, and availability. To mitigate this, the ability to insert FBX files has been disabled in Word, Excel, PowerPoint, Outlook, and 3D Viewer. The change is effective as of the January 9, 2024 security update. Microsoft has provided security updates for affected products, and affected customers can get the update from the Microsoft Store or through their organizations. Additionally, PowerShell command 'Get-AppxPackage -Name Microsoft.Microsoft3DViewer' can be used to check if the update is installed. The security update for Microsoft Office 2021 for Mac is not immediately available, and affected customers will be notified when it is released.
